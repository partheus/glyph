Tech

The site is, of course, built via Eleventy and hosted on Netlify.

HTML, CSS, Markdown, and Nunjucks templates are used to build the site.

The monospace font stack is as follows:

Menlo, Consolas, Monaco, Liberation Mono, Lucida Console, Andale Mono WT, Andale Mono, Lucida Sans Typewriter, DejaVu Sans Mono, Bitstream Vera Sans Mono, Nimbus Mono L, Courier, monospace;

No language models or tracking services are used on this site. I don't care about all that.
